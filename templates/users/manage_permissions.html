Permissions System Guide
  • Default Permissions: Each user role has a set of default permissions defined based on that role's duties.
  • Custom Permissions: System administrator can enable or disable specific permissions for each user. These permissions replace the default permissions.
  • Unauthenticated Users: Only have access to public permissions (view products and about us).
  • Role Change: When changing a user's role, their default permissions also change unless custom permissions are set.
